Freemarker的基本使用及入门

nana · 2019-7-25 · 次阅读

一 ： Freemarker 的概述

l FreeMarker 是一个用 Java 语言编写的模板引擎，它基于模板来生成文本输出。

l Freemarker可以用作表现层的实现技术，还可以用于生成 XML、Html、或 Java 等。

二 ：Freemarker 的基本使用步骤

2.1 项目基本结构

2.2 基本使用步骤

2.2.1 导入相关依赖 Freemarker依赖
<dependency>
        <groupId>org.freemarker</groupId>
        <artifactId>freemarker</artifactId>
        <version>2.3.23</version>
</dependency>
2.2.2 模板结构

模板文件中四种元素

l 文本：直接输出的部分

l 注释：即<#–…–>格式不会输出

l 插值（Interpolation）：即${..}部分,将使用数据模型中的部分替代输出

l FTL指令：FreeMarker指令，和HTML标记类似，名字前加#予以区分，不会输出。

2.2.3 创建模板文件
<html>
<head>
    <meta charset="utf-8">
    <title>Freemarker入门小DEMO </title>
</head>
<body>
<#--我只是一个注释，我不会有任何输出  -->
${name},你好。${message}
</body>
</html>
2.2.4 生成文件

使用步骤：

第一步：创建一个 Configuration 对象，直接 new 一个对象。构造方法的参数就是freemarker的版本号。

第二步：设置模板文件所在的路径。

第三步：设置模板文件使用的字符集。一般就是 utf-8.

第四步：加载一个模板，创建一个模板对象。

第五步：创建一个模板使用的数据集，可以是 pojo 也可以是 map。一般是 Map。

第六步：创建一个 Writer 对象，一般创建一 FileWriter 对象，指定生成的文件名。

第七步：调用模板对象的 process 方法输出文件。

第八步：关闭流

代码：

创建Test类 main方法如下：

  //1.创建配置类
        Configuration configuration=new Configuration(Configuration.getVersion());
        //2.设置模板所在的目录 
        configuration.setDirectoryForTemplateLoading(new File("D:/pinyougou_work/freemarkerDemo/src/main/resources/"));
        //3.设置字符集
        configuration.setDefaultEncoding("utf-8");
        //4.加载模板
        Template template = configuration.getTemplate("test.ftl");
        //5.创建数据模型
        Map map=new HashMap();
        map.put("name", "张三 ");
        map.put("message", "欢迎来到神奇的品优购世界！");
        //6.创建Writer对象
        Writer out =new FileWriter(new File("d:\\test.html"));
        //7.输出
        template.process(map, out);
        //8.关闭Writer对象
        out.close();

执行后，在D盘根目录即可看到生成的test.html ，打开看看

三 ：相关指令

3.1.1 assign 指令

此指令用于在页面上定义一个变量

  • 定义对象类型 ：
<#assign info={"mobile":"13301231212",'address':'北京市昌平区王府街'} >
电话：${info.mobile}  地址：${info.address}

3.3 空值处理运算符

如果你在模板中使用了变量但是在代码中没有对变量赋值，那么运行生成时会抛出异常。但是有些时候，有的变量确实是null，怎么解决这个问题呢？

3.3.1 判断某变量是否存在 “??”

用法为:variable??,如果该变量存在,返回true,否则返回false 

<#if aaa??>
  aaa变量存在
<#else>
  aaa变量不存在
</#if>
3.3.2 缺失变量默认值 ： “！”

我们除了可以判断是否为空值，也可以使用!对null值做转换处理

在模板文件中加入

  ${aaa!'-'}

在代码中不对aaa赋值，也不会报错了 ，当aaa为null则返回！后边的内容-

3.4 运算符

3.4.1 算数运算符

FreeMarker表达式中完全支持算术运算,FreeMarker支持的算术运算符包括:+, - , * , / , %

3.4.2 逻辑运算符

逻辑运算符有如下几个:

l 逻辑与：&&

l 逻辑或：||

l 逻辑非：!

注意：逻辑运算符只能作用于布尔值,否则将产生错误

3.4.3 比较运算符

表达式中支持的比较运算符有如下几个:

l =或者==：判断两个值是否相等.

l !=：判断两个值是否不等.

l >或者gt：判断左边值是否大于右边值

l >=或者gte：判断左边值是否大于等于右边值

l <或者lt：判断左边值是否小于右边值

l <=或者lte：判断左边值是否小于等于右边值

注意:

=和!=可以用于字符串,数值和日期来比较是否相等,

但=和!=两边必须是相同类型的值,否则会产生错误,

FreeMarker是精确比较,,”x “,”X”是不等的.

其它的运行符可以作用于数字和日期,但不能作用于字符串,

使用gt等字母运算符代替>会有更好的效果,因为 FreeMarker会把>解释成FTL标签的结束字符,当然,也可以使用括号来避免这种情况,如:<#if (x>y)>

4.1 FreemarkerTest 相关代码

package com.pinyougou.freemarker.test;

import freemarker.template.Configuration;
import freemarker.template.Template;

import java.io.File;
import java.io.FileWriter;
import java.io.IOException;
import java.io.Writer;
import java.util.*;

public class FreemarkerTest {

    public static void main(String[] args) throws Exception {
        //1.创建配置对象，设置版本号
        Configuration configuration = new Configuration(Configuration.getVersion());
        //2.设置加载模板的目录
        configuration.setDirectoryForTemplateLoading(new File("D:\\Work\\workspaces\\pinyougou\\pyg_91\\pyg_freemarker_demo\\src\\main\\resources\\"));
        //3.设置加载编码格式：utf-8
        configuration.setDefaultEncoding("utf-8");
        //4.从模板目录下加载模板文件
        Template template = configuration.getTemplate("test.ftl");
        //5.准备数据map
        Map map = new HashMap<String, String>();
        map.put("name","九纹龙");
        map.put("message","你真牛！");
        map.put("isMarketable",true);
        List<Map<String, String>> courseList = new ArrayList();
        Map coursemap = new HashMap<String, String>();
        coursemap.put("name","基础课程");
        coursemap.put("price",500);
        coursemap.put("days", 11);
        courseList.add(coursemap);
        coursemap = new HashMap<String, String>();
        coursemap.put("name","web课程");
        coursemap.put("price",7000);
        coursemap.put("days", 12);
        courseList.add(coursemap);
        coursemap = new HashMap<String, String>();
        coursemap.put("name","框架课程");
        coursemap.put("price",7000);
        coursemap.put("days", 15);
        coursemap = new HashMap<String, String>();
        coursemap.put("name","项目课程");
        coursemap.put("price",14000);
        coursemap.put("days", 40);
        courseList.add(coursemap);
        map.put("courseList", courseList);

        map.put("today", new Date());
//        map.put("abc", new Date());
        //6.将模板和数据整合输出到指定的位置
        Writer out = new FileWriter(new File("D:\\Work\\appdata\\htmls\\91\\test.html"));
        template.process(map, out);
        //7.关闭资源
        out.close();
    }
}
